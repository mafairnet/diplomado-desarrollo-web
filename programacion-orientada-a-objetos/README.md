Programacion Orientada A Objetos
===================
En este repositorio se encontrara el material de apoyo para el Diplomado de Desarrollo Web.

Acerca de
-----------
#### Objetivo: Demostrar los temas vistos en clase a traves de codigo.


Prerequisites
-----------
-PHP (5.4 en adelante)
-MySQL Server (5.1 en adelante)

Relacion Temario<->Archivos
-----------
This system needs 3 separate servers to work.
1. Free PBX
2. Oreka OpenSource
3. Aloha Server (with Dejavu (Audio Fingerprinting) Python Module)
